1️⃣ What is the difference between var, let, and const?
Ans: var can be redeclared and reassigned and hoisted with undefined, 
    let can not be redeclared but it can be reassigned,
    const can not be redeclared or reassigned and it must be initialized when declared.

2️⃣ What is the spread operator (...)?
Ans: Spread operator spreads elements of arrays or objects into indivisual elements, it makes easier to use data.

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans: map() creates a new array to each element,
     filter() creates a new array with the elements that matches condition
     forEach() it runs a function on each element and did not return a new array.
